# NotesApp

## Мобільний додаток для збереження та зміни нотатків

## Що нового?

- Збереження нотатків з використанням технології `Convex`
- Сучасніша стилізація
- Зміна статусу завдань та статистика
- Анімації при видаленні та створенні нотатків

## Встановлення

1. Клонуйте репозиторій
2. Встановіть залежності: `npm install`
3. Запустіть проєкт: `npx expo start`

4. ## Використані технології

- React Native
- TypeScript
- Expo
- Convex
